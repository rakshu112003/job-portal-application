Job Portal Frontend

A modern React-based frontend for the Job Portal application. It allows users to browse, search, filter, save, and manage job listings with an interactive and responsive user interface.

🚀 Live Demo

Frontend: https://job-portal-application-hnk8-phi.vercel.app/

✨ Features

- Responsive UI
- Add, Update and Delete Jobs
- Search Jobs by Title or Company
- Filter Jobs by City
- Sort Jobs
- Save/Favorite Jobs
- Dashboard Analytics (Charts)
- CSV Export
- Pagination
- Form Validation
- Loading Indicators
- Toast Notifications

🛠️ Tech Stack

- React
- Axios
- Recharts
- React CSV
- CSS

📦 Installation

git clone https://github.com/rakshu112003/job-portal-application.git
cd job-portal-application/frontend
npm install
npm start

🏗️ Build

npm run build
